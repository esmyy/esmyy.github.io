# 编辑器

`VS Code` 还是 `WebStorm` ？，想用啥用啥，不纠结。

用过 WebStorm

- 智能提示很强，不需要太多的配置就能用的比较爽
- 以前还有配置同步的便利，现在 VS Code 也有了
- logo 很好看， VS Code 的 logo 丑。

现在用 VS Code

- 身边同事用的多
- 足够好用
- 同步也已经十分方便
- snippets 配置很爽，WebStorm 的 Snippets 配置比较麻烦

## 插件配置
