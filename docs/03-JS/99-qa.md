`typeof` 还有一个比较特殊的地方，常见的变量安全检测不安全了，如下

```ts
typeof x === "undefined";
```

TDZ(暂时性死区) 的表现是 inaccessible，而 typeof 是一个 RHS，是尝试做一次 access 操作，GG...
