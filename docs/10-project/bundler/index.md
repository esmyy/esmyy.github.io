# 构建工具

所谓构建过程，就是拿源文件，做一堆转换，处理，最后输出编译后的文件。

```mermaid
  flowchart LR
  Input(源文件) --> Transform(转换) --> Output(输出)
```

以上这种解释往往被认为是一句废话，因为太过正确。

但也许某一天，千帆过尽，岁月斑驳，会在某一刻明悟，原来苦苦求索的道理，就在那太过正确的”废话“之中。
